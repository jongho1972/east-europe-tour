# 동유럽 여행 코스 2026 🗺️

2026년 여름 동유럽 여행 일정을 인터랙티브 지도로 보여주는 웹 앱입니다.

**배포 URL**: https://jhawk-east-europe-tour.netlify.app

## 여행 개요

- **기간**: 2026.7.3(목) ~ 7.13(일), 9박 11일
- **경로**: 부다페스트 → 비엔나 → 할슈타트 → 잘츠부르크 → 체스키크룸로프 → 프라하

| 구간 | 도시 | 일차 |
|------|------|------|
| 🇭🇺 헝가리 | 부다페스트 | D1~D3 |
| 🇦🇹 오스트리아 | 비엔나 · 할슈타트 · 잘츠부르크 | D4~D7 |
| 🇨🇿 체코 | 체스키크룸로프 · 프라하 | D8~D10 |

## 주요 기능

- Google Maps 기반 인터랙티브 일정 지도
- 날짜별 / 도시별 필터링 (사이드바 탭)
- 일자 클릭 시 해당 날짜 마커 및 경로선 하이라이트
- 마커 클릭 시 장소 정보 팝업
- 도시 간 이동 경로선 (실선: 직접 이동, 점선: 당일치기/셔틀)
- 모바일 반응형 레이아웃

## 접근 제한

가족 전용 페이지로 진입 시 오버레이형 비밀번호 게이트가 노출됩니다 (매 로드 재인증).

## 기술 스택

- HTML / CSS / JavaScript (단일 파일)
- Google Maps JavaScript API
- Netlify (GitHub 자동 배포)

## 스크린샷

| 데스크톱 | 모바일 |
|----------|--------|
| 사이드바 + 전체 지도 | 탭 전환 (일정 ↔ 지도) |
